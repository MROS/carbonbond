## vscode 建議安裝套件

### eslint

預設不會作用到 .ts, .tsx 的檔案，請在 vscode 的 setting.json 中加入

```
"eslint.validate": [
    "javascript",
    "javascriptreact",
    "typescript",
    "typescriptreact",
],
```

### postcss-sugarss-language

讓 vscode 讀懂 postcss 的語法。

爲使其能作用於 .css 檔，在 setting.json 中加入

```
"files.associations": {
    "*.css": "postcss"
},
```
